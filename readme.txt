Simple functionality for show and hide content eg. in a faq.

== Description ==
An easy way to show or hide content by clicking on a element.

Use it like this:
&lt;a class="ssh-question" href="#"&gt;Question&lt;/div&gt;
&lt;div class="ssh-answer ssh-hide"&gt;Answer&lt;/div&gt;

Make sure the answer div is the next element after the element with the question.

== Installation ==

Using the Plugin Manager

1. Click Plugins
2. Click Add New
3. Search for "simple show hide for wordpress"
4. Click Install
5. Click Install Now
6. Click Activate Plugin

Manually

1. Unzip simple-show-hide-for-wordpress.zip
2. Upload simple-show-hide-for-wordpress folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
